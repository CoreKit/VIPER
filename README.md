# VIPER interfaces

Essential interfaces (protocols) for the VIPER architecture.



## Installation

### SPM

```
.package(url: "https://github.com/CoreKit/VIPER", from: "2.0.0"),
```


### CocoaPods

```
source 'https://github.com/CoreKit/VIPER.git'
pod 'VIPER', '~> 2.0.0'
```

### Carthage

```
github "CoreKit/VIPER" "2.0.0"
```
